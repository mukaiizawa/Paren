# NAME
comparable - comparable type.

# DESCRIPTION
The comparable type refers to the following types to which the comparison function can call.

- bytes(7)
- keyword(7)
- number(7)
- string(7)
- symbol(7)

The comparison functions are as follows.

- <(3)
- <=(3)
- >(3)
- >=(3)
