# NAME
sequence - sequence type.

# DESCRIPTION
The types listed below are called `sequences`.

- array(7)
- bytes(7)
- list(7)
- string(7)

The functions in the following list are supported by sequence.

- concat(3)
- last(3)
- slice(3)
- sort!(3)
- swap!(3)

Everything except string is mutable.
