# NAME
sequence - sequence type.

# DESCRIPTION
A sequence is one of the following types.

- array(7)
- bytes(7)
- list(7)
- string(7)

Everything except string is mutable.

All of the following functions are applicable to sequences.

- concat(3)
- last(3)
- slice(3)
- sort!(3)
- swap!(3)
