# NAME
number - number data type.

# DESCRIPTION
A number data type represent numeric value.

Integers and real numbers support the range that can be represented by the `int64_t` and `double` types of the C language, respectively.

# EXAMPLES

    ) 1
    1
    ) 1.3
    1.3
    ) (+ 1 2 3)
    6
