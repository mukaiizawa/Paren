# NAME
keyword - make a keyword.

# SYNOPSIS

    (keyword NAME)

# DESCRIPTION
The function `keyword` make a keyword.

# RETURN VALUE
Returns a keyword consisting of the bytes-like object `NAME`.

# EXAMPLES

    ) (keyword "foo")
    :foo
