# NAME
address - object address.

# SYNOPSIS

    (address X)

# DESCRIPTION
The function `address` returns a address of `X`.

# RETURN VALUE
Returns a address.

# NOTES
The fact that the two objects `X` and `Y` have the same address is a necessary and sufficient condition for `(== X Y)` to evaluate to true.
