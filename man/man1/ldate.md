# NAME
ldate - list date.

# SYNOPSIS
    ldate [OPTION] [[YEAR] MONTH]

# DESCRIPTION
Enumerates dates for a given month.

If the year and month are omitted, it is considered that the current year and month are specified.

# OPTIONS

    -l
        Use long listing format.

# SEE ALSO
- `cal(1)`
- `date(1)`
