# NAME
ls - list directory contents.

# SYNOPSIS

    ls [OPTION]... [PATH]

# DESCRIPTION
List information about the PATH (the current directory by default).

# OPTIONS

    -l
        Use long listing format.
    -r
        List subdirectories recursively.
    -f
        List only file.
    -d
        List only directories.
    -F
        Use full path format.
