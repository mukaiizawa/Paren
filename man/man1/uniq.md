# NAME
uniq - report or omit repeated lines.

# SYNOPSIS

    uniq [OPTION]...

# DESCRIPTION
Print the unique lines in the standard input.

# OPTIONS

    -c
        Prefix lines by the number of occurrences.
    -d
        Only print duplicate lines.
    -u
        Only print unique lines.

# NOTES
The input must be sorted.

# SEE ALSO
- `sort(1)`
