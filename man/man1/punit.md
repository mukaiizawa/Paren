# NAME
punit - unit test for paren.

# SYNOPSIS

    punit [OPTION]... [DIR ...]

# DESCRIPTION
Run unit tests for the specified `DIRs` (the current directory by default).

# OPTIONS

    -i FILE [, FILE]...
        Ignore files.
    -r
        Test subdirectories recursively.

# SEE ALSO
- `manunit(1)`
